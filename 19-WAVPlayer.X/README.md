# WAV Player

Playback of Audio (WAV) files from SD Card (microSD Click© board)

Need:

* Device: PIC16F18855 (or other PIC16F1 with 8/10-bit PWM and SPI/MSSP)
* Board: XPRESS Evaluation Board (DM164140)
* microSD Click © board
*

## The Basic Idea
1. Connect a microSD ClickÂ© board  (3V)
2. Launch MPLAB X and MPLAB Code Configurator to quickly initialise all the peripherals
3. Play music/voice at 22050 Hz ()8-bit mono)...


## Limitations
* Play 8-bit mono files only, with extension WAV
